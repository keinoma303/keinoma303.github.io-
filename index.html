Microsoft Copilot (一般向け)

Windowsやウェブブラウザで利用できる一般的なAIアシスタント

主な特徴

  • 利用場所: ウェブブラウザ（copilot.microsoft.com）やWindows 11のタスクバーから直接アクセス可能
  • 基本機能: 質問応答、文章生成、情報検索、画像生成（DALL-E）、テキスト分析
  • 連携サービス: Bing検索エンジンと連携し、最新の情報にアクセス可能
  • マルチモーダル対応: テキスト入力だけでなく、画像アップロードからの質問や音声入力にも対応
  • 複数言語サポート: 日本語を含む多言語に対応し、翻訳機能も搭載
  • 料金: 基本機能は無料で利用可能（画像生成などの高度な機能は制限あり）

詳細情報: Microsoft Copilot を始めましょう

Microsoft Copilot for Windows

Windows 11に組み込まれたAIアシスタント

主な特徴

  • 統合環境: Windows 11のオペレーティングシステムに深く統合されたAIアシスタント
  • アクセス方法: タスクバーのCopilotアイコンまたはWin+Cキーボードショートカットで起動
  • OSレベルの操作: Windowsの設定変更やシステム操作をAIの支援で実行可能
  • アプリ連携: 画面上の情報を認識し、コンテキストに応じた支援を提供
  • スクリーンショット解析: 画面のスクリーンショットを撮影し、内容について質問や操作が可能
  • Windows設定の最適化: パフォーマンス改善やバッテリー設定など、使用状況に応じた最適化提案

詳細情報: Microsoft Copilot for Windows 公式情報

Microsoft Copilot Pro

より高度な機能を備えた有料版サービス

主な特徴

  • 高性能AI: 最新かつ高性能なAIモデル（GPT-4o）への優先アクセス
  • 生成AI機能強化: 高品質な画像生成（DALL-E 3）の1日あたりの生成数が100枚まで増加
  • Microsoft 365連携: Word、Excel、PowerPoint、Outlookなどのアプリ内でCopilot機能が利用可能
  • 優先利用権: 需要の高い時間帯でも優先的にサービスを利用可能（レスポンス速度向上）
  • 高度なプロンプト機能: より複雑な指示や長文の処理が可能
  • GPTs作成機能: 特定の目的に特化したカスタムCopilotを作成可能
  • Designers機能: より高度なデザイン生成と編集機能

料金: 月額2,000円（税別）

注意点: Microsoft 365アプリ内での利用には、Microsoft 365 Personal（1,284円/月）または Family（1,284円/月）のサブスクリプションも別途必要

詳細情報: Copilot Pro の使用を開始する

Microsoft Copilot for Microsoft 365

Wordやexcel、PowerPointなどのMicrosoft 365アプリと統合されたビジネス向けバージョン

主な特徴

  • アプリ統合: Word、Excel、PowerPoint、Outlook、Teams、SharePoint、OneDriveなどのMicrosoft 365アプリ内で直接利用可能
  • 企業データ連携: 組織内のデータやドキュメント、SharePointサイト、Teams会話を参照してコンテキストを理解
  • セキュリティ対応: 企業のセキュリティポリシーやコンプライアンス要件に準拠したデータ保護機能
  • チーム連携: 会議の要約や議事録作成、タスク管理などをAI支援
  • 大規模言語モデル: 最先端のGPT-4モデルを活用し、複雑な業務タスクを処理
  • 管理機能: IT管理者向けの展開・管理・監視ツール
  • コンプライアンス: GDPR、HIPAA、ISO 27001などの国際規格に準拠

料金: 企業向けライセンス（ユーザーあたり月額約3,000円〜）

前提条件: Microsoft 365 E3/E5、Business Standard/Premium、またはOffice 365 E3/E5のライセンスが必要

詳細情報: Microsoft 365 Copilot で新しい生産性を引き出しましょう
